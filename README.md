# HardTest

## Tech stack

### Backend:
- NodeJS v16.17.0
- ExpressJS v4.18.1
- Prisma v4.3.1
- MySQL v8.0
- Typescript

### Frontend
- Angular v14.2.3
- Typescript

## Instructions

### Backend instructions
- Installation `npm install`
- Database setup
  - `cd src`
  - `npx prisma migrate dev --name init`
  - `npx prisma generate`
- Start server
  - `npm run develop`

### Frontend instructions
- Installation `npm install`
- Start server
  - `npm run start`
- Port: **4200**
